# App Controle Álbum Copa 2026

## Rodar

```bash
pip install streamlit pandas
streamlit run app.py
```

## Uso

- Escolha grupo e time na lateral.
- Amarelo = já tem.
- Branco = falta.
- Consulta rápida: digite ou use o microfone do teclado do celular: `Brasil 10`.

## Ajustar dados

Edite `album_data.json`.
